Data

We populated our database tables using Columbia University's Million Song Dataset. https://labrosa.ee.columbia.edu/millionsong/

Entity-Relationship Diagram

Entity Set: song, artist, tags
Relationship Set: artist-tag, similar (artist-artist), song-artist

Database Schema

artist: id, name, familiarity, hotness
song: track_id, title, release (album name), duration, year
tag: id, tag
applies: artist_id, tag_id
sings: track_id, artist_id
similar_to: artist1_id, artist2_id
